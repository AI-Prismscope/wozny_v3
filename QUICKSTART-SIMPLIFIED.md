# Wozny v3 - Quickstart Guide
## Three Ways to Use Wozny (Simple → Advanced)

---

## 🟢 Level 1: Out of the Box (2 minutes)
**Perfect for: First-time users, quick data cleaning**

### The Simplest Path
1. **Upload** your CSV file
2. **Accept** the automatic schema detection
3. **View** your data report automatically
4. **Go to Workshop** to see your data
5. **Export** your cleaned file

### What Happens Automatically
- ✅ Data quality analysis
- ✅ Health score calculation
- ✅ Duplicate detection
- ✅ Missing value identification
- ✅ Type detection

### No Configuration Needed
- Schema is auto-detected
- Best practices applied automatically
- Smart defaults for everything
- Just upload and go!

**Result:** Clean data in 2 minutes with zero configuration.

---

## 🟡 Level 2: Light Customization (5 minutes)
**Perfect for: Users who want some control, recurring workflows**

### Your Path
1. **Upload** your CSV file
2. **Review** the schema suggestion
3. **Adjust** if needed (change data type)
4. **Accept** and view report
5. **Use AI features** in Workshop:
   - Ask Wozny questions
   - Apply smart filters
   - Use ML grouping for similar values
6. **Export** with your preferences

### What You Can Adjust
- ✏️ Data type classification (Customer/Transaction/Inventory/Time-Series)
- ✏️ Column types and formats
- ✏️ Duplicate detection rules
- ✏️ Export format preferences

### Light AI Usage
- **Ask Wozny**: "Show me rows with missing emails"
- **Smart Filters**: Natural language queries
- **ML Grouping**: Group similar company names
- **Smart Split**: Separate combined columns

**Result:** Tailored cleaning with minimal effort.

---

## 🔴 Level 3: Full Control (15+ minutes)
**Perfect for: Power users, complex datasets, custom workflows**

### Your Path
1. **Create custom rules** (before upload)
   - Define your own schema detection rules
   - Set up templates for recurring data types
   - Configure duplicate detection logic
2. **Upload** your CSV file
3. **Review** classification (uses your custom rules)
4. **Fine-tune** in Workshop:
   - Manual data editing
   - Custom transformations
   - Advanced filtering
   - Batch operations
5. **Use advanced AI**:
   - Complex natural language queries
   - Multi-column operations
   - Semantic analysis
   - Pattern detection
6. **Export** with full control

### Full Customization Options
- 🎛️ Custom schema classification rules
- 🎛️ Priority-based rule matching
- 🎛️ Template creation for recurring data
- 🎛️ Batch processing multiple files
- 🎛️ Advanced duplicate detection
- 🎛️ Custom export formats
- 🎛️ Offline model configuration

### Advanced AI Features
- **Complex Queries**: "Find all transactions from Q4 2024 where amount > $1000 and status is pending"
- **Batch Operations**: Apply changes to multiple columns at once
- **Pattern Learning**: Train on your specific data patterns
- **Custom Grouping**: Define your own similarity rules

### Power User Tools
- Custom rule templates (9 pre-built available)
- Analytics dashboard
- Classification history
- Rule priority management
- Batch file processing
- Export/import configurations

**Result:** Complete control over every aspect of data cleaning.

---

## 🔌 Working Offline (All Levels)

### First Time (One-Time Setup)
1. **Connect to internet**
2. **Upload any CSV** (triggers model download)
3. **Wait 2-3 minutes** for models to cache (~50MB)
4. **See "Ready" indicator**

### After Setup
- ✅ Disconnect internet anytime
- ✅ All features work normally
- ✅ Models cached in browser
- ✅ 100% private, local processing

**Works offline at all levels!**

## 🎯 Which Level Should You Choose?

### Choose Level 1 if:
- ✓ First time using Wozny
- ✓ Simple, straightforward CSV
- ✓ Just need basic cleaning
- ✓ Want results immediately
- ✓ Trust automatic detection

### Choose Level 2 if:
- ✓ Comfortable with the basics
- ✓ Need some AI assistance
- ✓ Want to filter/search data
- ✓ Have specific data type needs
- ✓ Recurring but simple workflows

### Choose Level 3 if:
- ✓ Complex or unusual data formats
- ✓ Need custom detection rules
- ✓ Processing many similar files
- ✓ Require advanced AI features
- ✓ Want complete control
- ✓ Building reusable workflows

## 💡 Pro Tips by Level

### Level 1 Tips
- Start here! You can always move to Level 2 later
- The auto-detection is very accurate
- Export early and often

### Level 2 Tips
- Use "Ask Wozny" for complex filters instead of manual searching
- ML grouping saves hours on messy data
- Adjust schema if your data type is misclassified

### Level 3 Tips
- Create templates for recurring data types
- Use high priority (90+) for very specific rules
- Export your custom rules for backup
- Check analytics to optimize your rules

## 🚀 Getting Started Right Now

### Absolute Beginner?
→ **Start with Level 1**
1. Click "Upload CSV"
2. Drop your file
3. Click "Accept" when schema appears
4. Go to Workshop
5. Click "Export"

Done! 🎉

### Have 5 More Minutes?
→ **Try Level 2**
1. After upload, review the schema
2. Click "Ask Wozny" in Workshop
3. Try: "Show me rows with missing data"
4. Use ML grouping on messy columns
5. Export

### Ready for Power User Mode?
→ **Jump to Level 3**
1. Go to Settings → Custom Rules
2. Create a rule or use a template
3. Upload your CSV
4. Explore Analytics
5. Set up your perfect workflow
