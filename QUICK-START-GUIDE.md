# Wozny v3 - Quick Start Guide
## Three Ways to Use Wozny (Simple → Advanced)

Choose your path based on your needs and experience level.

---

## 🟢 Level 1: Out of the Box
**Time: 2 minutes | Perfect for: First-time users, quick cleaning**

### The Simplest Workflow
```
Upload CSV → Accept Schema → View Report → Workshop → Export
```

### Steps
1. **Upload** - Drag & drop your CSV file
2. **Accept** - Click "Yes" when schema is detected
3. **Report** - Automatically see your data quality report
4. **Workshop** - Click "Go to Workshop" to view your data
5. **Export** - Click "Export" to download cleaned file

### What Happens Automatically
✅ Data quality analysis  
✅ Health score (0-100)  
✅ Duplicate detection  
✅ Missing value identification  
✅ Column type detection  

### Zero Configuration
- No settings to adjust
- No rules to create
- No AI prompts needed
- Just upload and go!

**Result: Clean data in 2 minutes**

---

## 🟡 Level 2: Light Customization
**Time: 5 minutes | Perfect for: Some control, recurring workflows**

### Enhanced Workflow
```
Upload CSV → Review Schema → Adjust → Report → Use AI → Export
```

### Steps
1. **Upload** - Drag & drop your CSV file
2. **Review Schema** - Check the auto-detected data type
3. **Adjust** (optional) - Change to Customer/Transaction/Inventory/Time-Series
4. **Accept** - Proceed to report
5. **Workshop** - Use AI features:
   - Ask Wozny: "Show me rows with missing emails"
   - Smart Filters: Natural language queries
   - ML Grouping: Group similar values like "Google" and "Google Inc"
   - Smart Split: Separate combined columns
6. **Export** - Download with your preferences

### What You Control
✏️ Data type classification  
✏️ Column formats  
✏️ AI-powered filtering  
✏️ Semantic grouping  
✏️ Export preferences  

### AI Features Available
- **Ask Wozny**: Natural language queries
  - "Find duplicate emails"
  - "Show rows where status is pending"
  - "Which columns have missing data?"
- **ML Grouping**: Automatically group similar values
- **Smart Split**: Intelligently separate combined data

**Result: Tailored cleaning with AI assistance**

---

## 🔴 Level 3: Full Control
**Time: 15+ minutes | Perfect for: Power users, complex data, custom workflows**

### Advanced Workflow
```
Create Rules → Upload CSV → Fine-tune → Advanced AI → Batch Process → Export
```

### Steps
1. **Create Custom Rules** (before upload)
   - Go to Settings → Custom Rules
   - Use templates or create from scratch
   - Define schema detection logic
   - Set priority levels
2. **Upload** - Your custom rules apply automatically
3. **Fine-tune** - Manual adjustments in Workshop:
   - Edit individual cells
   - Custom transformations
   - Advanced filtering
   - Multi-column operations
4. **Advanced AI**:
   - Complex queries: "Find Q4 2024 transactions > $1000 where status is pending"
   - Pattern detection across columns
   - Semantic analysis
   - Batch operations
5. **Batch Process** - Handle multiple files at once
6. **Export** - Full control over format and options

### Full Customization
🎛️ Custom schema classification rules  
🎛️ Priority-based rule matching  
🎛️ Template creation (9 pre-built available)  
🎛️ Batch file processing  
🎛️ Advanced duplicate detection  
🎛️ Analytics dashboard  
🎛️ Classification history  

### Power User Tools
- **Custom Rules**: Create reusable detection patterns
- **Templates**: Pre-built rules for common data types
- **Batch Processing**: Handle multiple files simultaneously
- **Analytics**: Track classification trends and accuracy
- **Export/Import**: Share configurations across machines

**Result: Complete control over every aspect**

### First-Time Setup (One-Time, 2-3 minutes)
1. Connect to internet
2. Click "Load Models" button (in About page offline section)
3. AI models download automatically (~50MB)
4. Wait for "Ready" indicator
5. Disconnect from internet
6. Upload your CSV and proceed

### After Setup
✅ Disconnect internet anytime  
✅ All features work normally  
✅ Models cached in browser  
✅ 100% private, local processing  

**All three levels work offline!**

## ❓ Frequently Asked Questions

### "Which level should I start with?"
**Always start with Level 1.** Even experienced users should try the simplest path first. You'll be surprised how well automatic detection works.

### "Can I switch between levels?"
**Yes!** You can start at Level 1 and move to Level 2 or 3 anytime. Each level builds on the previous one.

### "Do I need to configure anything for offline mode?"
**No.** Just upload a file while online (first time only). Models download automatically and you're set for offline use forever.

### "How accurate is the automatic schema detection?"
**Very accurate** for common data types (Customer, Transaction, Inventory, Time-Series). For unusual formats, use Level 3 custom rules.

### "What's the difference between 'Ask Wozny' and manual filtering?"
**Ask Wozny** understands natural language ("Show me rows with missing emails") while manual filtering requires you to know exact column names and values.

### "Can I save my custom rules?"
**Yes** (Level 3). Export your rules as JSON and import them on other machines or share with your team.

### "Is my data really private?"
**100% Yes.** Everything runs in your browser. No data is ever uploaded to servers. Works completely offline after initial setup.

### "What file size can Wozny handle?"
**Up to 100MB** CSV files. For larger files, consider splitting them or using batch processing (Level 3).
